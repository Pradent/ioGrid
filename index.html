1. Overview

We wanted to build a CSS framework which offered much more flexibility towards the elements you will use than the ones you might use. ioGrid is a lightweight, non-intrusive, production-friendly CSS Framework.

1.1 Why ioGrid

Lightweight

ioGrid is lightweight, we have kept our focus only on the core elements for a Webpage.

Flexible

ioGrid is fairly flexible, it provides various utility classes and uses Flexbox as well.

Prefixable

Already have an Webpage/app and think ioGrid will interfere with other elements? Just prefix every class in ioGrid with a few clicks!

1.2 Usage

npm install ioGrid

1.3 Sample Page

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">

    <title>ioGrid Barebones</title>

    <link href="css/ioGrid.min.css" rel="stylesheet">

  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

6. Utilities

Category Class Use
Visibility
.hidden Hides an element by setting display to none.
.visible Show an element by setting display to block.
.hidden--soft Hides an element by setting visibility to hidden and opacity to 0.
.visible--soft Show an element by setting visibility to visible and opacity to 1.
Alignment
.block--center Center align a block element.
.block--right Right align a block element.
.text--left Left align the text.
.text--center Center align the text.
.text--right Right align the text.
.text--right Right align the text.
List Styling
.list--unstyled Removes bullets or numbers from a list.
.list--inline Removes bullets or numbers from a list and the list items in an inline row.
Resets
._nomt Sets margin-top to 0
._nomr Sets margin-right to 0
._nomb Sets margin-bottom to 0
._noml Sets margin-left to 0
._nopt Sets padding-top to 0
._nopr Sets padding-right to 0
._nopb Sets padding-bottom to 0
._nopl Sets padding-left to 0
